1 Kelengkapan section Website ini terdiri dari 5 section, yaitu : Menu, Header, Content, Sidebar, dan Footer.
2 Kelengkapan jumlah halaman Website ini memiliki 4 halaman : halaman pertama berisi konten website, halaman kedua berisi formulir pemesanan kamar hotel, halaman ketiga berisi evaluasi website (ringkasan), dan halaman keempat berisi rincian tim website dan pembagian tugas.
3 Kesesuian nama file untuk halaman utama File "index.html" untuk halaman utama telah dimodifikasi untuk memenuhi persyaratan Tugas Aplikasi.
4 Kelengkapan penggunaan elemen HMTL form yang digunakan sebagai field masukan Website kami menggunakan 6 field masukan, yaitu : textbox, radio, checkbox, dropdown menu, textarea, dan submit button.
5 Kelengkapan penerapan validasi form Website ini menggunakan 6 validasi form : required, alfabet, numerik, email, panjang digit dan karakter tertentu, serta tanggal.
6 Kesesuaian pesan error validasi form
  • Jika field inputan tidak diisi, maka muncul pesan error "Field tidak boleh kosong". Jika
  • field nama terdapat angka, maka pesan errornya "Field harus berisi huruf" atau jika
  • field nomor telepon terdapat huruf, maka pesan errornya "Nomor telepon harus berupa angka", dan apabila
  • Check-In Check-Out maka muncul pesan error seperti "Tanggal check-out tidak valid".
7 Konsistensi desain antar halaman Website ini sudah sesuai antar halamannya, mulai dari penggunaan font, warna, serta layout dari halaman-webnya.
8 Kesesuaian website terhadap tema Website ini sesuai dengan tema yang telah diberikan.
